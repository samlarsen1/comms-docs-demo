# Money

Monetary amount with ISO 4217 currency code. Amount is represented as a string to avoid floating point precision issues. Aligned with ISO 20022 ActiveOrHistoricCurrencyAndAmount.

## Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `amount` | `string` | Yes | — | Decimal monetary amount as a string, e.g. '1234.56'. Negative values use a leading minus, e.g. '-50.00'. |
| `currency` | `string` | Yes | — | ISO 4217 three-letter currency code, e.g. 'GBP', 'EUR', 'USD'. |
