Contact information for a party. Aligned with ISO 20022 ContactDetails2. Phone numbers should use E.164 format (e.g. '+447911123456').

## Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `emailAddress` | `string` _(optional)_ | No | `null` | Primary email address for electronic communications. |
| `mobileNumber` | `string` _(optional)_ | No | `null` | Mobile phone number in E.164 format, used for SMS and push notifications. |
| `homePhoneNumber` | `string` _(optional)_ | No | `null` | Home landline phone number in E.164 format. |
| `workPhoneNumber` | `string` _(optional)_ | No | `null` | Work phone number in E.164 format. |
| `preferredContactMethod` | `PreferredContactMethod` _(inline enum)_ | No | `"EMAIL"` | Customer's preferred method of direct contact. |
