# CommercialStatementPayload

| | |
|---|---|
| **Namespace** | `com.bank.schemas.payloads.statements` |
| **Type** | `record` |
| **Version** | `1.0.0` |
| **Compatibility** | `BACKWARD` |
| **Status** | `ACTIVE` |
| **Source** | [schemas/payloads/statements/CommercialStatementPayload.avsc](https://github.com/samlarsen1/comms-docs-schemas/blob/main/schemas/payloads/statements/CommercialStatementPayload.avsc) |

Payload for a commercial or business banking account statement. Covers current accounts, commercial loans, and trade finance facilities. Aligned with UK Finance commercial banking standards and Basel III reporting requirements.

## Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `statementId` | `string` | Yes | — | Unique identifier for this statement. |
| `accountId` | `string` | Yes | — | Reference to the commercial Account. |
| `accountNumber` | `string` | Yes | — | Account number. |
| `sortCode` | `string` _(optional)_ | No | `null` | Sort code for UK accounts. |
| `iban` | `string` _(optional)_ | No | `null` | IBAN if available. |
| `companyName` | `string` | Yes | — | Legal name of the business account holder. |
| `companyRegistrationNumber` | `string` _(optional)_ | No | `null` | Companies House registration number. |
| `productName` | `string` | Yes | — | Account product name, e.g. 'Business Current Account', 'Commercial Loan'. |
| `productType` | [`ProductType`](./common/enums/ProductType) | Yes | — | Commercial product classification. |
| `relationshipManagerName` | `string` _(optional)_ | No | `null` | Name of the assigned relationship manager. |
| `relationshipManagerContact` | `string` _(optional)_ | No | `null` | Direct contact number or email for the relationship manager. |
| `statementDate` | `string` | Yes | — | ISO 8601 date the statement was generated. |

| `interestChargedThisPeriod` | [`Money`](./common/Money) _(optional)_ | No | `null` | Total interest charged during the period. |
