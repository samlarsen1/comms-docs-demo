# InterestRateChangeEmailPayload

| | |
|---|---|
| **Namespace** | `com.bank.schemas.payloads.emails` |
| **Type** | `record` |
| **Version** | `1.0.0` |
| **Compatibility** | `BACKWARD` |
| **Status** | `ACTIVE` |
| **Source** | [schemas/payloads/emails/InterestRateChangeEmailPayload.avsc](https://github.com/samlarsen1/comms-docs-schemas/blob/main/schemas/payloads/emails/InterestRateChangeEmailPayload.avsc) |

Payload for an email notification of an interest rate change. Digital channel equivalent of InterestRateChangeLetterPayload. Omits postal address fields and includes email-specific fields for rendering.

## Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `notificationId` | `string` | Yes | — | Unique identifier for this email notification instance. |
| `referenceNumber` | `string` | Yes | — | Bank reference number for this notice. |
| `recipientEmailAddress` | `string` | Yes | — | The email address this notification will be sent to. Must match a verified address on the customer's profile. |
| `recipientName` | `string` | Yes | — | Recipient's preferred name for personalisation, e.g. 'John'. |
| `accountId` | `string` | Yes | — | Reference to the affected Account. |
| `accountReference` | `string` | Yes | — | Displayable account reference or masked account number. |
| `productName` | `string` | Yes | — | Name of the product whose rate is changing. |
| `productType` | [`ProductType`](./common/enums/ProductType) | Yes | — | Product type of the affected account. |
| `rateChangeReason` | `com.bank.schemas.common.enums.RateChangeReason` | Yes | — | Reason for the interest rate change. |
| `currentInterestRatePercent` | `string` | Yes | — | Current annual interest rate as a decimal string. |
| `newInterestRatePercent` | `string` | Yes | — | New annual interest rate as a decimal string. |
| `rateType` | [`InterestRateType`](./common/enums/InterestRateType) | Yes | — | Structure of the new interest rate. |
| `effectiveDate` | `string` | Yes | — | ISO 8601 date from which the new rate takes effect. |
| `impactOnMonthlyPayment` | `com.bank.schemas.payloads.letters.PaymentImpact` _(optional)_ | No | `null` | Impact on monthly payments if applicable. |
| `ctaUrl` | `string` _(optional)_ | No | `null` | Call-to-action URL for the email, e.g. link to online banking to view account details. |
| `ctaText` | `string` _(optional)_ | No | `null` | Call-to-action button text, e.g. 'View your account'. |
| `unsubscribeUrl` | `string` _(optional)_ | No | `null` | URL for the customer to manage email preferences. Required for marketing communications. |
